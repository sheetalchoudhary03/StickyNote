# 📝 Sticky Notes Desktop Application

## 📌 Introduction
The Sticky Notes project is a lightweight desktop application developed using **C++** and the **Qt framework** on Ubuntu. It allows users to create and manage quick notes directly on their desktop, helping improve productivity without relying on heavy applications.

This project was developed as part of an **Operating Systems** course to gain practical understanding of how applications are built, compiled, and executed in a Linux environment.

---

## 🎯 Objective
- Design and develop a simple sticky notes application using C++ and Qt  
- Understand project structure including source files, headers, object files, and makefiles  
- Gain hands-on experience with compilation, linking, and execution in Ubuntu  
- Provide users with a quick and easy way to manage short notes  

---

## ✨ Features
- Create and manage sticky notes on desktop  
- Lightweight and fast performance  
- Simple and user-friendly interface  
- Runs efficiently on Linux (Ubuntu)  

---

## 🛠️ Tech Stack
- **Language:** C++  
- **Framework:** Qt  
- **Platform:** Ubuntu (Linux)
